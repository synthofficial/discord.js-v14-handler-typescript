## 🎯 Prerequisites
To be able to run this project, run the following commands in your terminal.

```shell
npm i -g pnpm
pnpm install
```

## ⚒️ Build
To build this project, you can run the below command.

```shell
npm run build
```
or to live test this project you can run
```shell
npm run dev
```
